## COMO INICIAR UM PROJETO

- Importar os swcs das bibliotecas necessárias listadas abaixo;
- Deixar a Main.as como Application Class do projeto;
- alterar todas as entradas de "project_name" pelo nome do projeto;

#Bibliotecas necessárias:

- saci http://saci.googlecode.com/svn/trunk/src/saci/
- com.adobe.serialization.json.JSON http://saci.googlecode.com/svn/trunk/external-libs/adobe.json.swc (original em http://as3corelib.googlecode.com/svn/trunk/src/com/adobe?r=83)
- net.hires.debug.Stats http://saci.googlecode.com/svn/trunk/external-libs/hires.stats.swc (original em http://mrdoob.googlecode.com/svn/trunk/libs/net/hires/debug/?r=144)
- redneck http://saci.googlecode.com/svn/trunk/external-libs/redneck.swc (original em )
- br.com.stimuli.BulkLoader http://saci.googlecode.com/svn/trunk/external-libs/stimuli.bulkloader.swc (original em http://bulk-loader.googlecode.com/svn/trunk/src/br/com/stimuli/loading/?r=250)
- com.asual.swfaddress http://saci.googlecode.com/svn/trunk/external-libs/swfaddress.swc (original em https://swfaddress.svn.sourceforge.net/svnroot/swfaddress/trunk/swfaddress/dist/as/3/com/asual/swfaddress/)
